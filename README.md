### 실행방법

```
git clone
cd open-market
npm i
npm start
```

### 구현사항

- 노래 구매 목록 페이지 구현
- 노래 상세페이지 구현
- 상세페이지 내에서 옵션,갯수 선택 가능
- 옵션과 아이템 선택 갯수에 따라 상품의 총액 계산 노출
- 장바구니 담기 구현
- 아이템 상세에서 장바구니에 담기를 클릭하면 상단에 구현된 장바구니 아이콘 옆에 담긴 상품 갯수 노출
- 장바구니 목록 페이지 구현
- 장바구니 목록에는 아이템명과 아이템의 옵션, 수량, 개별금액, 총 금액 노출
- 총 3개의 페이지 구현 → 목록 / 상세 / 장바구니
- 금액은 노출될때 천원단위 콤마 표시
- 상품목록 정렬기능
